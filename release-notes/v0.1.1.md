## Highlights

- AAC: full profile coverage (LC, Main, SSR, LTP, HE, HEv2)
- Error handling: detailed ADTS validation diagnostics and actionable failures
- APIs: MP4E-compatible helpers (`new_with_fragment()`, `flush()`, `set_create_time()`, `set_language()`)
- Codec support: HEVC/H.265 (Annex B + VPS/SPS/PPS), AV1 (OBU stream + Sequence Header), Opus (raw packets)
- CLI tool improvements and property-based invariant testing framework

## Highlights

This release focuses on VP9 completeness, API modernization, and CLI diagnostics.

- VP9 production readiness: full-range flag parsing and corrected color metadata (`vpcC`)
- API modernization: removed deprecated constructors; standardized construction through `MuxerBuilder`
- Enhanced `muxide info`: improved codec/audio detection for troubleshooting
- Documentation/testing alignment and cleanup for a pristine release posture
